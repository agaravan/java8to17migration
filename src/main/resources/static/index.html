Migration Strategy

1

Clone & Analyze

Clone the repository and analyze pom.xml files, Java source code, and dependency tree for compatibility issues.

2

Configure OpenRewrite

Inject the OpenRewrite Maven plugin with appropriate recipes based on project analysis (JAXB, Lombok, Spring, etc.).

3

Apply Recipes

Execute the appropriate UpgradeToJava{version} and supporting recipes to transform source code and build files.

4

Report & Review

Generate a comprehensive migration report with issues found, changes made, and manual steps remaining.
